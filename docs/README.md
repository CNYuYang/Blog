## MapReduce

使用 Java 编写的一个分布式计算框架，本项目为 MapReduce 的最简实现，未引入任何第三方库， Master 能保证线程安全的向 Worker 分配 Map、 Reduce 任务，通过 Socket 完成 Rpc 通信
