第 1 章 概述

JAX-RS 2.0 (JSR-339) 和 JAX-RS 2.1 (JSR-370) 属于JCP 规范。JCP 规范是在 HTTP 协议基础上提供 RESTful 网站服务的Java API。RESTEasy 是这些规范的可移植实现，可以在任何 Servlet 容器中运行。还可以与 WildFly 应用程序服务器进行更紧密的集成，从而使该环境中的用户体验更好。RESTEasy 还在普通 JAX-RS 功能之上提供了额外的特性。
