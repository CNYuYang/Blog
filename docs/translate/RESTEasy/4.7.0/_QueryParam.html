第 6 章 @QueryParam

注意

RESTEasy @QueryParam 注解支持不直接指定参数名。

@QueryParam 注解允许您将URI查询字符串参数或url表单编码的参数映射到您的方法调用。

GET /books?num=5 

@GET
public String getBooks(@QueryParam("num") int num) {
...
}

当前，由于RESTEasy是基于Servlet构建的，因此无法区分URI查询字符串还是url 表单的编码参数。 与PathParam一样，您的参数类型可以是String，原生类型，String作为输入参数的构造函数或静态 valueOf() 方法。
