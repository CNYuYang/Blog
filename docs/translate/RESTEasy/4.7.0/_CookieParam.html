第 10 章 @CookieParam

注意

RESTEasy @CookieParam 注解支持不直接指定参数名。

@CookieParam 注释允许您将 cookie 的值或 HTTP 请求 cookie 的对象表示注入到方法调用中

GET /books?num=5 

@GET
public String getBooks(@CookieParam("sessionid") int id) {
...
}

@GET
public String getBooks(@CookieParam("sessionid") javax.ws.rs.core.Cookie id) {...}

与 PathParam 类似，您的参数类型可以是任何基元类型、 String 或任何具有接受 String 参数的构造函数的任何 Java 对象，或者接受 String 作为参数的静态 valueOf 方法。还可以通过 javax.ws.rs.core.Cookie 类获得 cookie 的对象表示。
