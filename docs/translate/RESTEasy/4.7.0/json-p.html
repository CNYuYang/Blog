第 25 章 通过Java EE 7 JSON-P API 支持JSON

不，这不是你所想的 JSONP！JSON-P 是一个新的 Java EE 7 JSON 解析 API。对于一个新的 JSON 解析 API 来说，这个名字太可怕了！他们在想什么？不管怎样，RESTEasy 有一个供应商。如果你正在使用 WildFly，它是 Java EE 7 所要求的，所以你会自动捆绑它。否则，使用这个 maven 依赖项。 

<dependency>
   <groupId>org.jboss.resteasy</groupId>
   <artifactId>resteasy-json-p-provider</artifactId>
   <version>4.7.0-SNAPSHOT</version>
</dependency>

它内置了对 JsonObject、 JsonArray 和 JsonStructure 作为请求或响应实体的支持。如果你的路径上也有这样的程序，那就不应该与Jackson发生冲突。
