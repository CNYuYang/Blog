前言

RESTEasy JAX-RS 的商业开发支持、生产支持和商业培训都可以通过联系JBoss 获得，JBoss 是 Red Hat 公司的一个部门(见 http://www.JBoss.com/)。

在一些示例演示中，本应该在一行中内容，但是由于页面宽度的限制，将他们分成几行表示。行尾的'\'表示为了适应页面宽度而引入的一个分隔符，它表示该行还没结束。所以: 

Let's pretend to have an extremely \
long line that \
does not fit
This one is short

真正想表示的是: 

Let's pretend to have an extremely long line that does not fit
This one is short
